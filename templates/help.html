Hades v2.0 help page

  • Hades V2.0 is a web application built by the corporate security online team
  • Can be Used by multiple users to categorise online adverts found by Hades
  • Automatically send suspected cases and take down cases to polonius case mangement
  • Remembers the last category selected for a seller
  • Coming soon ! version will have machine learning to suggest if advert crop protection or seeds

Hades v2 Categories: What do they mean?

Category Action Explanation
Suspected Counterfeiter A case is opened in Polonious A test purchase will be conducted for the case that is opened in Polonious
Takedown A case is opened in Polonious The case that is opened in Polonious, followed by a takedown of the advert
Not Relevant Deletes all adverts for that seller for that e-commerce site Applies to all adverts placed by the seller in the future. No adverts will be seen from this seller
Syngenta Authorised Whitelist adverts by this seller New adverts will not appear for review, but will be kept in the database
No Action Deletes this particular advert A particular advert will be deleted, but other adverts from the seller will appear for review
